# Shared Resources

This directory contains cross-cutting resources, configurations, and utilities shared across all applications and modules.

## Purpose

To prevent code duplication and enforce consistency across different components of the Sapana Live Tracker project. It includes:

- **Constants**: Shared configurations, HTTP status structures, and domain boundaries.
- **Types**: Global TypeScript interfaces, standard tracking payloads, and database schema representations.
- **Formatting Utilities**: Common helpers for geographic coords, time zones, distance measurements, and speed calculations.
