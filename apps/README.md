# Applications

This directory contains the user-facing application projects for the Sapana Live Tracker ecosystem.

## Directory Structure

- `worker-mobile/`: The Capacitor-powered mobile application used by field workers for background GPS tracking and offline data gathering.
- `admin-web/`: The web dashboard used by supervisors and administrators for real-time tracking, route playback, and worker management.

These applications compose the shared modules and engines to deliver specific end-user experiences.
